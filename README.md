## 📂 Estructura del Repositorio

Asegúrese de estar visualizando la rama **`master`** en GitHub para acceder a la totalidad de los archivos del proyecto:

*   `/src` - Componentes y lógica principal de la aplicación en React.
*   `/public` - Archivos públicos y assets del proyecto.
*   `package.json` - Dependencias y scripts de configuración.
